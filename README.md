# Intersect

Open Source Software Powering Intersect API

## Setup

1. Install the latest version of [Bun](https://bun.sh/)
2. Run `bun install`

## Running Web App

1. Run `bun run dev:web`
2. Navigate to `localhost:3000`

## Running Storybook

1. Run `bun run dev:ui`
2. Navigate to `localhost:6006`
